# SunDevPro — Impression serveur centralisée V1.0.23

Les postes clients ne cherchent plus une imprimante Windows locale. Ils interrogent le poste serveur SunDevPro, récupèrent la liste des imprimantes du serveur et lui transmettent les travaux d'impression.

Livrables prévus par GitHub Actions :
- `SunDevPro_V1.0.23_SETUP_COMPLET.exe`
- `SunDevPro_V1.0.23_MISE_A_JOUR.exe`

Le serveur utilise le port LAN TCP 17823. L'accès doit rester limité au réseau privé Windows.
